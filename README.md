# How to use

```sh
yarn create react-app myapp --typescript
cd myapp
# Make sure your python3 is >= 3.6
curl -sSL https://raw.githubusercontent.com/oursky/create-react-app-oursky/master/bootstrap.py | python3 - .
git diff # To see what were changed
```

- browserslist
- prettier
- eslint
- stylelint
